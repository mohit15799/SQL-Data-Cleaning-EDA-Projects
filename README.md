# SQL Data Cleaning & Exploratory Data Analysis Projects

This repository contains SQL projects demonstrating real-world data cleaning and exploratory data analysis (EDA) techniques using MySQL.

## Skills Demonstrated

- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Joins
- CTEs
- Window Functions
- Aggregate Functions
- CASE Statements
- Date Functions
- String Functions
- Temporary Tables
- Views

## Projects

### 1. Data Cleaning Project

Objective:
Clean raw data by removing duplicates, handling missing values, standardizing formats, correcting inconsistencies, and preparing the dataset for analysis.

Key Techniques:
- Remove Duplicates
- Standardize Data
- Handle NULL Values
- Trim Spaces
- Convert Date Formats

Layoffs Dataset

### 2. Exploratory Data Analysis Project

Objective:
Analyze cleaned data to uncover trends, identify patterns, and generate business insights.

Key Analysis:
- Total Layoffs by Company

- Layoffs by Country
- Monthly Layoff Trends
- Top Companies by Layoffs
- Rolling Totals
- Ranking Companies
